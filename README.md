# Userscripts

A repository for all the userscripts I've written.

## Usage

For the scripts that are still maintained, the easiest way is to just install them directly using the buttons below.

| Script                    | Description                                                                                           | Install                                                                                                                                                                                       |
| ------------------------- | ----------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| One Click Copy on YouTube | Replaces the YouTube share button with a copy button that copies the shortlinks using a single click. | [![badge](https://img.shields.io/badge/Install-green?style=for-the-badge&logo=tampermonkey)](https://github.com/SinTan1729/userscripts/raw/refs/heads/main/one-click-copy-on-youtube.user.js) |

There are also some old scripts in the [discontinued directory](discontinued).

Alternatively, you can use [Greasy Fork](https://greasyfork.org/en/users/293284-sayantan-santra) for the installation.
